# Streaming Setup Wizard (MVP)

Interactive setup flow for the uscardforum post:
- Topic: https://www.uscardforum.com/t/topic/221571
- Goal: guide users from zero to successful playback in one flow.

## Scope (MVP)
- 6-step guided wizard
- Platform split guidance
- RD token input (not persisted by default)
- Config summary card
- Basic troubleshooting branch

## Run
Open `app/index.html` in a browser.

## Notes
- This MVP is static front-end only.
- Sensitive fields (like token) are masked and not saved unless user opts in.
